## Дополнительные Исследование по программированию

### Цель
Разобраться в работе Redis. Исследовать как меняется погрешность курса Доллара к Рублю
в зависимости от частоты обновления кэша в Redis. Сравнить насколько Redis 
быстрее чем просто запрашивать курс через api Центр Банка.

### Реализация
Я создал класс ```ManagerApi```, создал команду ```TestRedis, TestWithRedis, TestWithNotRedis```. С помощью данных
команд можно исследовать поставленные нам задачи
